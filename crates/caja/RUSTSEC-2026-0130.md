# Out-of-bounds access in `Index` and `IndexMut` implementations

The `Index` and `IndexMut` implementations for `Caja` use pointer arithmetic
without any bounds checking. Creating a `Caja` with a small key and then
accessing an out-of-range index allows accesses beyond beyond the bounds of the
allocated memory.

This can be triggered without the use of unsafe code via the `[]` indexing
operator.
