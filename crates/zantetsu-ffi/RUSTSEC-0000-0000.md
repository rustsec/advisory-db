# zantetsu-ffi is unmaintained

The `zantetsu-ffi` crate is no longer maintained. The Node.js, Python, and C
FFI bindings it provided were removed as part of the zantetsu 0.2 release,
which refocused the project on its core Rust library.

A tombstone version (0.2.0) has been published and 0.1.4 has been yanked.
There is no replacement for `zantetsu-ffi`.

## Migration

If you were using `zantetsu-ffi` for anime filename parsing, no direct
equivalent is available. If your project is Rust-based, switch to
[`zantetsu 0.2`](https://crates.io/crates/zantetsu).
