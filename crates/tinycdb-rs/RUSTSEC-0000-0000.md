# tinycdb-rs is unmaintained

tinycdb-rs will no longer be maintained as the last commit was 11 years ago. The project has an unsoundness issue, which can further lead to a memory safety bug as in [issue #6](https://github.com/andrew-d/tinycdb-rs/issues/6).
