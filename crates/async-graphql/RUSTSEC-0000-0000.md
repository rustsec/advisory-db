# Directive Overload

async-graphql before 7.0.10 does not limit the number
of directives for a field. This can lead to Service Disruption,
Resource Exhaustion, and User Experience Degradation.

Use SchemaBuilder.limit_directives to limit the maximum number
of directives for a single field.
