# `sui-execution-cut` was removed from crates.io for malicious code

`sui-execution-cut` included a build script that attempted to exfiltrate data
from the build machine.

The malicious crate had 1 version published on 2026-04-20 and had no evidence
of actual usage. This crate had no dependencies on crates.io.
