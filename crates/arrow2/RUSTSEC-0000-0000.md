            self.buffer.get_unchecked(start..end)

Struct Rows and its row_unchecked is public accessible, and row_unchecked does not mark with unsafe. However, it does not have enough check to the row, which might cause memory risks, under a safe API. In Rust, we are not expecting memory risks if merely using safe API.
